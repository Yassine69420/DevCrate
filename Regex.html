Regex Cheatsheet

Anchors

^ - Start of string

$ - End of string

\b - Word boundary

Character Classes

\d - Digit (0-9)

\w - Word character (a-z, A-Z, 0-9, _)

\s - Whitespace character

. - Any character (except newline)

Quantifiers

* - 0 or more

+ - 1 or more

? - 0 or 1

{n} - Exactly n times

Groups & Ranges

(...) - Capturing group

[abc] - Match a, b, or c

[a-z] - Any lowercase letter

Flags

g - Global match

i - Case-insensitive

m - Multiline mode

Escaped Characters

\+ - Match a literal '+' sign

\. - Match a literal '.'

\\ - Match a literal '\'
